Hello. If you're seeing this page it's because you are currently using a browser that doesn't support WebP. This includes Safari below version 15.
Unfortunately, that version of Safari and below does not support webp images, which are crucial to the map, panorama and various assets.
I explored adding regular pngs, but the cost was too high to host them (the map is 1/4 the size and the panoramas are 1/10 the size as webp images.)
Please update your browser or alternatively use a modern version of Chrome or Firefox to play.
